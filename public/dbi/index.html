DBI

Start by first connecting to the database, Drivers section describes how to accomplish that.

List tables

dbListTables(con)
## [1] "airlines" "airport"  "airports" "faithful" "flights"  "iris"

Upload data to the database

library(nycflights13)

data(package = "nycflights13")

dbWriteTable(con, "airlines", airlines)

List Fields

dbListFields(con, "iris")
## [1] "Sepal.Length" "Sepal.Width"  "Petal.Length" "Petal.Width" 
## [5] "Species"

Run a query

Using the appropriate SQL syntax for the type of connection, you can use the dbGetQuery run queries.

dbGetQuery(con, "SELECT TOP 10 * from airports")
